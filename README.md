# Dropouts Turf — Official Website

Website for **Dropouts Turf**, Vasai's biggest FIFA certified grass turf.

## 📁 Files
- `index.html` — Main HTML page
- `style.css`  — All styles
- `script.js`  — Countdown timer, mobile nav, scroll effects
- `logo.png`   — Brand logo

## 🚀 How to Host on GitHub Pages

1. **Create a new GitHub repository** (e.g. `dropouts-turf`)
2. Upload all 4 files: `index.html`, `style.css`, `script.js`, `logo.png`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at:
   `https://<your-github-username>.github.io/dropouts-turf`
